# serp-analyzer

競合サイトのGoogle検索流入数を自動推定するWebツール。

## 何ができるか

URLを入れるだけで以下を全自動で実行:

1. 競合サイトのHTMLを取得してキーワードを自動抽出
2. 各キーワードのGoogle検索結果から、対象サイトの順位を取得
3. 順位別CTRと推定検索数から月間Google流入数を推定

シングルURL分析と、複数URL比較（最大5サイト）に対応。

## 仕組み

- **キーワード抽出**: title, meta description, meta keywords, h1〜h3 から重み付きスコアで上位8件を抽出
- **順位取得**: User-Agentランダム化 + 1〜3秒のランダム待機 + ブロック検知時のリトライで、Google検索を直接スクレイピング
- **検索数推定**: キーワードの文字数・一般用語含有率から月間検索ボリュームを概算
- **流入計算**: 順位別CTR（Advanced Web Rankingデータ参考） × 推定検索数 ÷ 捕捉率30%

## 制約・注意

- 推定値です。誤差2〜3倍の可能性あり
- Google直接スクレイピング方式のため、1日20〜30サイトが安全圏
- 1サイト分析に20〜40秒かかる（待機を入れてブロックを回避するため）
- 商用大規模利用ではScraperAPI/SerpAPI等の有料サービス推奨
